<h2>מה הולך כאן?</h2>
בשעה טובה הגענו אל אחת מאבני היסוד של האלגברה המופשטת - המשפט היסודי של תורת גלואה. כדי להבין מה המשפט אומר צריך להכיר את המושגים הבסיסיים שהוא מתעסק בהם - <a href="https://gadial.net/2018/04/03/field_extensions_intro/">הרחבת שדות</a> ו<a href="https://gadial.net/2018/04/26/galois_theory_intro/">אוטומורפיזמים של שדות</a> והשדה שאוטומורפיזם משמר. מכירים? יופי, אז הנה המשפט:

תהא {% equation %}E/F{% endequation %} הרחבת גלואה ו-{% equation %}G=\text{Gal}\left(E/F\right){% endequation %} חבורת הגלואה של ההרחבה. אז קיימת התאמה חד-חד-ערכית ועל בין <strong>שדות הביניים</strong> {% equation %}K{% endequation %} המקיימים {% equation %}F\subseteq K\subseteq E{% endequation %} ובין <strong>תתי-חבורות</strong> {% equation %}H\subseteq G{% endequation %}. ההתאמה מוגדרת כך: בכיוון אחד, בהינתן {% equation %}K{% endequation %}, היא מחזירה את כל האיברים של {% equation %}G{% endequation %} שמשמרים את {% equation %}K{% endequation %}; בכיוון השני, בהינתן {% equation %}H{% endequation %}, היא מחזירה את כל האיברים של {% equation %}E{% endequation %} שמשתמרים על ידי {% equation %}H{% endequation %}.

יותר מכך: ההתאמה שומרת על מבנה ה<strong>סריג</strong> של תתי-השדות של {% equation %}E{% endequation %} שמכילים את {% equation %}F{% endequation %} ותתי-החבורות של {% equation %}G{% endequation %}, תוך שהיא הופכת את הסדר. כלומר, עבור תתי-שדות {% equation %}K_{1},K_{2}{% endequation %} ותתי-החבורות שמתאימים להם {% equation %}H_{1},H_{2}{% endequation %}:
<ul>
 	<li>אם {% equation %}K_{1}\subseteq K_{2}{% endequation %} אז {% equation %}H_{2}\subseteq H_{1}{% endequation %}.</li>
 	<li>תת-החבורה שמתאימה לשדה {% equation %}\left\langle K_{1},K_{2}\right\rangle {% endequation %} שנוצר על ידי {% equation %}K_{1},K_{2}{% endequation %} היא {% equation %}H_{1}\cap H_{2}{% endequation %}.</li>
 	<li>תת-החבורה שמתאימה לשדה {% equation %}K_{1}\cap K_{2}{% endequation %} היא תת-החבורה {% equation %}\left\langle H_{1},H_{2}\right\rangle {% endequation %} שנוצרת על ידי {% equation %}H_{1},H_{2}{% endequation %}.</li>
</ul>
יותר מכך: לכל {% equation %}F\subseteq K\subseteq E{% endequation %} כזו אנחנו יודעים ש:
<ul>
 	<li>{% equation %}E/K{% endequation %} הרחבת גלואה עם חבורת גלואה {% equation %}\text{Gal}\left(E/K\right)=H{% endequation %}. לכן בפרט {% equation %}\left[E:K\right]=\left|H\right|{% endequation %} ו-{% equation %}\left[K:F\right]=\left|G:H\right|{% endequation %} (האינדקס של {% equation %}H{% endequation %} ב-{% equation %}G{% endequation %}).</li>
 	<li>{% equation %}K/F{% endequation %} היא הרחבת גלואה אם ורק אם {% equation %}H{% endequation %} היא תת-חבורה <strong>נורמלית </strong>ב-{% equation %}G{% endequation %} ובמקרה זה, {% equation %}\text{Gal}\left(K/F\right)\cong G/H{% endequation %}.</li>
</ul>
אני רוצה להסביר את הקטע עם הסריג קצת יותר בפירוט, אבל קודם - תמונות. כאן תמונה אחת באמת שווה אלף מילים.

{% equation %}\xymatrix{ &amp; \mathbb{Q}\left(\sqrt{2},\sqrt{3}\right)\ar[dl]^{2}\ar[d]^{2}\ar[dr]^{2}\\ \mathbb{Q}\left(\sqrt{2}\right)\ar[dr]^{2} &amp; \mathbb{Q}\left(\sqrt{6}\right)\ar[d]^{2} &amp; \mathbb{Q}\left(\sqrt{3}\right)\ar[dl]^{2}\\ &amp; \mathbb{Q} } {% endequation %}

{% equation %}\xymatrix{ &amp; \left\{ e\right\} \ar[dl]^{2}\ar[d]^{2}\ar[dr]^{2}\\ \left\{ e,\left(1,0\right)\right\} \ar[dr]^{2} &amp; \left\{ e,\left(1,1\right)\right\} \ar[d]^{2} &amp; \left\{ e,\left(0,1\right)\right\} \ar[dl]^{2}\\ &amp; \left\{ e,\left(1,0\right),\left(0,1\right),\left(1,1\right)\right\} } {% endequation %}

מה אנחנו רואים פה? בתמונה הראשונה יש לנו את ההרחבה {% equation %}\mathbb{Q}\left(\sqrt{2},\sqrt{3}\right)/\mathbb{Q}{% endequation %} (שדה הפיצול של הפולינום {% equation %}\left(x^{2}-2\right)\left(x^{2}-3\right){% endequation %} מעל {% equation %}\mathbb{Q}{% endequation %} עם כל תת-השדות שלה. המספרים ליד החצים מתארים את המימדים של ההרחבות. למשל, {% equation %}\left[\mathbb{Q}\left(\sqrt{2},\sqrt{3}\right):\mathbb{Q}\left(\sqrt{6}\right)\right]=2{% endequation %}.

בתמונה השניה יש לנו את החבורה {% equation %}\mathbb{Z}_{2}\times\mathbb{Z}_{2}{% endequation %} עם כל תת-החבורות שלה. שימו לב שאני מצייר את החבורה ותתי-החבורות שלה בסדר <strong>הפוך</strong>: תת חבורה נמצאת <strong>מעל</strong> תתי-החבורות שהיא מוכלת בהן. גם המשמעות של המספרים השתנתה: אם יש לנו חץ מתת-החבורה {% equation %}H{% endequation %} אל תת-החבורה {% equation %}G{% endequation %} (כלומר, {% equation %}H\subseteq G{% endequation %}) אז המספר הוא <strong>האינדקס</strong> של {% equation %}H{% endequation %} ב-{% equation %}G{% endequation %}, מה שאני מסמן בתור {% equation %}\left|G:H\right|{% endequation %} ושווה למספר הקוסטים של {% equation %}H{% endequation %} ב-{% equation %}G{% endequation %} ובמקרה שלנו, הסופי, גם פשוט ל-{% equation %}\frac{\left|G\right|}{\left|H\right|}{% endequation %}.

ברגע שבו הייתה לי הדיאגרמה הראשונה, היה קל לצייר את השניה - רק שיניתי את השמות של השדות כך שבמקומם יהיו תתי-חבורות מתאימות. את החצים עצמם והמספרים עליהם לא הייתי צריך לשנות. זו המהות של המשפט היסודי - המבנה של החצים ("סריג", ועוד אסביר במפורש מהו בהמשך) הוא זהה לחלוטין עד כדי שינוי שמות - <strong>איזומורפי</strong>.

עוד דוגמא? עוד דוגמא. ראינו גם את עלילות שדה הפיצול של {% equation %}x^{3}-2{% endequation %}. השורשים של הפולינום הזה היו {% equation %}\sqrt[3]{2},\omega\sqrt[3]{2},\omega^{2}\sqrt[3]{2}{% endequation %}, כאשר {% equation %}\omega=e^{\frac{2\pi i}{3}}{% endequation %} הוא <strong>שורש יחידה פרימיטיבי</strong> מסדר 3 - מספר מרוכב שמקיים {% equation %}\omega^{3}=1{% endequation %} והחזקות שלו הן כל הפתרונות המרוכבים של המשוואה {% equation %}x^{3}=1{% endequation %}. כל תמורה אפשרית של השורשים הללו נותנת אוטומורפיזם של שדה הפיצול, כך שחבורת הגלואה המתאימה היא {% equation %}S_{3}{% endequation %} - חבורת כל התמורות על שלושה איברים. איך החבורה הזו נראית, אנחנו יודעים טוב: היא כוללת את תמורת היחידה {% equation %}e{% endequation %}, את החילופים {% equation %}\left(1\ 2\right),\left(1\ 3\right),\left(2\ 3\right){% endequation %} ואת המעגלים מאורך 3 {% equation %}\left(1\ 2\ 3\right){% endequation %} ו-{% equation %}\left(1\ 3\ 2\right){% endequation %}. כל חילוף יוצר תת-חבורה מסדר 2 שכוללת את {% equation %}e{% endequation %} ועצמו; התמורה {% equation %}\left(1\ 2\ 3\right){% endequation %} יוצרת תת-חבורה מסדר 3 שכוללת גם את {% equation %}\left(1\ 3\ 2\right){% endequation %}; וזהו. אלו תתי-החבורות הלא טריוויאליות היחידות. אם מנסים למשל ליצור תת-חבורה מ-{% equation %}\left(1\ 2\right){% endequation %} ו-{% equation %}\left(1\ 3\right){% endequation %} אז המכפלה שלהם נותנת גם את {% equation %}\left(1\ 2\ 3\right){% endequation %} ו-{% equation %}\left(1\ 3\ 2\right){% endequation %} ומכאן כבר אפשר לקבל את כל {% equation %}S_{3}{% endequation %}. אם כן, את מבנה הסריג של {% equation %}S_{3}{% endequation %} אפשר לצייר כך:

{% equation %}\xymatrix{ &amp; \left\{ e\right\} \ar[ddl]^{3}\ar[d]^{2}\ar[dr]^{2}\ar[drr]\\ &amp; \left\{ e,\left(1\ 2\right)\right\} \ar[dd]^{3} &amp; \left\{ e,\left(1\ 3\right)\right\} \ar[ddl]^{3} &amp; \left\{ e,\left(2\ 3\right)\right\} \ar[ddll]^{3}\\ \left\{ e,\left(1\ 2\ 3\right),\left(1\ 3\ 2\right)\right\} \ar[dr]^{2}\\ &amp; S_{3} } {% endequation %}

מה השדות שמשתמרים על ידי תת-החבורות המתאימות? ובכן, {% equation %}\left\{ e,\left(1\ 2\right)\right\} {% endequation %} מחליפה שניים מהשורשים - נאמר ש"1" מתאר את {% equation %}\sqrt[3]{2}{% endequation %} ואילו "2" מתאר את {% equation %}\omega\sqrt[3]{2}{% endequation %}, אז {% equation %}\left(1\ 2\right){% endequation %} מחליפה אותם ולא משנה את {% equation %}\omega^{2}\sqrt[3]{2}{% endequation %}, כך שנקבל את שדה השבת {% equation %}\mathbb{Q}\left(\omega^{2}\sqrt[3]{2}\right){% endequation %} ומשהו דומה יקרה עם שתי החבורות האחרות מסדר 2. ומה עם החבורה מסדר 3? היא כוללת תמורות שמזיזות את כל השורשים, אז אף שורש לא יהיה שייך לשדה השבת שלה. אם כך, מי כן יהיה שם? הנה תעלול שיאפשר לנו לראות את זה בקלות: התמורה {% equation %}\sigma=\left(1\ 2\ 3\right){% endequation %} מבצעת את הדברים הבאים:

{% equation %}\sigma\left(\sqrt[3]{2}\right)=\omega\sqrt[3]{2}{% endequation %}

{% equation %}\sigma\left(\omega\sqrt[3]{2}\right)=\omega^{2}\sqrt[3]{2}{% endequation %}

ולכן אם <strong>נחלק</strong> את האיברים הללו האחד בשני נקבל:

{% equation %}\sigma\left(\omega\right)=\sigma\left(\frac{\omega\sqrt[3]{2}}{\sqrt[3]{2}}\right)=\frac{\omega^{2}\sqrt[3]{2}}{\omega\sqrt[3]{2}}=\omega{% endequation %}

באופן דומה, {% equation %}\tau=\left(1\ 3\ 2\right){% endequation %} מבצעת:

{% equation %}\tau\left(\sqrt[3]{2}\right)=\omega^{2}\sqrt[3]{2}{% endequation %}

{% equation %}\tau\left(\omega\sqrt[3]{2}\right)=\sqrt[3]{2}{% endequation %}

ולכן:

{% equation %}\tau\left(\omega\right)=\tau\left(\frac{\omega\sqrt[3]{2}}{\sqrt[3]{2}}\right)=\frac{\sqrt[3]{2}}{\omega^{2}\sqrt[3]{2}}=\omega^{-2}=\omega{% endequation %}

המעבר האחרון נובע מכך ש-{% equation %}\omega^{3}=1{% endequation %}.

אם כן, {% equation %}\omega{% endequation %} בוודאי <strong>שייך</strong> לשדה ש-{% equation %}\left\{ e,\left(1\ 2\ 3\right),\left(1\ 3\ 2\right)\right\} {% endequation %} משמרת. מכיוון שהפולינום המינימלי של {% equation %}\omega{% endequation %} מעל {% equation %}\mathbb{Q}{% endequation %} הוא {% equation %}x^{2}+x+1{% endequation %} (כי {% equation %}x^{3}-1=\left(x-1\right)\left(x^{2}+x+1\right){% endequation %}) אז מימד ההרחבה {% equation %}\mathbb{Q}\left(\omega\right)/\mathbb{Q}{% endequation %} הוא 2, בהתאם לאינדקס של {% equation %}\left\{ e,\left(1\ 2\ 3\right),\left(1\ 3\ 2\right)\right\} {% endequation %} ב-{% equation %}S_{3}{% endequation %}, כך שכבר ברור שזה כל שדה השבת, וקיבלנו את דיאגרמת השדות:

{% equation %}\xymatrix{ &amp; \mathbb{Q}\left(\sqrt[3]{2},\omega\right)\ar[ddl]^{3}\ar[d]^{2}\ar[dr]^{2}\ar[drr]\\ &amp; \mathbb{Q}\left(\omega^{2}\sqrt[3]{2}\right)\ar[dd]^{3} &amp; \mathbb{Q}\left(\omega\sqrt[3]{2}\right)\ar[ddl]^{3} &amp; \mathbb{Q}\left(\sqrt[3]{2}\right)\ar[ddll]^{3}\\ \mathbb{Q}\left(\omega\right)\ar[dr]^{2}\\ &amp; \mathbb{Q} } {% endequation %}

והמשפט היסודי של תורת גלואה גם מבטיח לנו שאלו כל שדות הביניים שיש. זו בעצם התכונה החשובה של המשפט היסודי: הוא הופך את הבעיה הקשה מאוד של מציאת שדות ביניים של הרחבות סופיות לבעיה הקצת פחות קשה של להבין את המבנה של חבורות סופיות.

עכשיו, תת-החבורה {% equation %}\left\{ e,\left(1\ 2\right)\right\} {% endequation %} היא לא נורמלית ב-{% equation %}S_{3}{% endequation %} (היא לא סגורה להצמדה; הצמדה של {% equation %}\left(1\ 2\right){% endequation %} מחזירה את {% equation %}\left(1\ 3\right){% endequation %} ו-{% equation %}\left(2\ 3\right){% endequation %}). זה אומר לנו שאם נסתכל על ההרחבה {% equation %}\mathbb{Q}\left(\omega^{2}\sqrt[3]{2}\right)/\mathbb{Q}{% endequation %}, היא לא תהיה הרחבת גלואה (וזה לא מפתיע - זו הרחבה שכוללת שורש אחד של הפולינום {% equation %}x^{3}-2{% endequation %} אבל לא את היתר). לעומת זאת, {% equation %}\left\{ e,\left(1\ 2\ 3\right),\left(1\ 3\ 2\right)\right\} {% endequation %} כן נורמלית (היא מאינדקס 2; כל תת-חבורה מאינדקס 2 היא נורמלית) ואכן {% equation %}\mathbb{Q}\left(\omega\right)/\mathbb{Q}{% endequation %} היא הרחבת גלואה (הרי היא שדה הפיצול של {% equation %}x^{2}+x+1{% endequation %} הספרבילי). בקיצור, נראה לי שהבנו - המשפט עובד!

בואו נסביר רק את הקטע של ה"סריג" הזה. המושג הבסיסי פה הוא <strong>קבוצה סדורה חלקית</strong>: זוג {% equation %}\left(P,\le\right){% endequation %} של קבוצה {% equation %}P{% endequation %} ויחס בינארי {% equation %}\le{% endequation %} מעליה הוא קבוצה סדורה חלקית אם {% equation %}\le{% endequation %} הוא רפלקסיבי, טרנזיטיבי ואנטי-סימטרי. היחס {% equation %}\subseteq{% endequation %} של שדות מקיים את זה, וגם {% equation %}\supseteq{% endequation %} של חבורות מקיים את זה. ההיפוך של הסימון לא מקרי: במקרה של חבורות, אני אומר ש-{% equation %}G\le H{% endequation %} אם ורק אם {% equation %}G\supseteq H{% endequation %}.

שתי קבוצות סדורות חלקית {% equation %}\left(P_{1},\le_{1}\right){% endequation %} ו-{% equation %}\left(P_{2},\le_{2}\right){% endequation %} הן <strong>איזומורפיות</strong> אם יש ביניהן העתקה חח"ע ועל {% equation %}f:P_{1}\to P_{2}{% endequation %} כך ש-{% equation %}a\le b\iff f\left(a\right)\le f\left(b\right){% endequation %}. המשמעות של איזומורפיזם כזה הוא ששתי הקבוצות הן <strong>אותו דבר בדיוק עד כדי שינוי שמות</strong> בכל הנוגע ליחסי הסדר בין האיברים שלהן. המשפט היסודי של תורת גלואה אומר (בין היתר) ששתי הקבוצות הסדורות שתיארתי - שדות ביניים של {% equation %}E/F{% endequation %} עם הכלה רגילה, ותתי-חבורות של {% equation %}\text{Gal}\left(E/F\right){% endequation %} עם הכלה הפוכה - הן איזומורפיות. לכן הציור שלהן יהיה אותו הציור בדיוק.

ומה עניין ה"סריג" כאן? ובכן, <strong>סריג</strong> במתמטיקה זו מילה שמתארת כל מני דברים, ואחד מהם הוא קבוצה סדורה חלקית {% equation %}\left(P,\le\right){% endequation %} כך שלכל שני איברים {% equation %}a,b\in P{% endequation %}, לקבוצה {% equation %}\left\{ a,b\right\} {% endequation %} יש חסם עליון וחסם תחתון. חסם עליון הוא איבר {% equation %}c{% endequation %} כך ש-{% equation %}a\le c{% endequation %} וגם {% equation %}b\le c{% endequation %} (למי שמקיים את שני אלו קוראים <strong>חסם מלעיל</strong> של הקבוצה) וכמו כן לכל {% equation %}d{% endequation %} שגם הוא חסם מלעיל, {% equation %}c\le d{% endequation %}. חסם תחתון זה אותו הדבר רק עם היפוך הכיוון של אי השוויונות. כדי שההגדרה הזו תהיה מעניינת בכלל צריך שיחס הסדר על {% equation %}P{% endequation %} לא יהיה מלא - כלומר, שיהיו איברים {% equation %}a,b{% endequation %} לא מתקיים לא {% equation %}a\le b{% endequation %} וגם לא {% equation %}b\le a{% endequation %} (כי אם למשל {% equation %}a\le b{% endequation %} אז החסם העליון של {% equation %}\left\{ a,b\right\} {% endequation %} הוא {% equation %}b{% endequation %} והתחתון הוא {% equation %}a{% endequation %}). הדוגמא הקלאסית לסריג היא המספרים הטבעיים עם יחס הסדר "{% equation %}a{% endequation %} מחלק את {% equation %}b{% endequation %}". שם החסם העליון של כל זוג איברים הוא ה-{% equation %}\text{lcm}{% endequation %} שלהם (הכפולה המשותפת המינימלית) והחסם התחתון הוא ה-{% equation %}\text{gcd}{% endequation %} שלהם (המחלק המשותף המקסימלי).

במקרה שלנו, עבור שני שדות {% equation %}\left\{ K_{1},K_{2}\right\} {% endequation %} ויחס ההכלה, החסם העליון הוא {% equation %}\left\langle K_{1},K_{2}\right\rangle {% endequation %} והתחתון הוא {% equation %}K_{1}\cap K_{2}{% endequation %}, ועם חבורות זה ההפך כי יחס הסדר הוא הפוך. מכאן ברורה ההתאמה בין השניים: השדה שמתאים לחסם העליון של {% equation %}\left\{ K_{1},K_{2}\right\} {% endequation %} <strong>צריך</strong> להתאים לחסם התחתון של {% equation %}\left\{ H_{1},H_{2}\right\} {% endequation %} עבור החבורות המתאימות {% equation %}H_{1},H_{2}{% endequation %}.
<h2>ואיך מוכיחים את זה?</h2>
את רוב הרכיבים של ההוכחה כבר יש לנו, ורק החלק עם חבורת המנה יהיה קשה. נתחיל מההתאמה עצמה. אני לוקח הרחבת גלואה {% equation %}E/F{% endequation %} ומסמן {% equation %}G=\text{Gal}\left(E/F\right){% endequation %}. לכל תת-חבורה {% equation %}H\subseteq G{% endequation %} אנחנו מתאימים את השדה {% equation %}\Psi\left(H\right)=\left\{ a\in E\ |\ \forall\sigma\in H:\sigma a=a\right\} {% endequation %} שמשתמר על ידי {% equation %}H{% endequation %}. כבר ראינו שזה אכן שדה וש-{% equation %}F\subseteq\Psi\left(H\right)\subseteq E{% endequation %}, כך שיש לנו <strong>פונקציה</strong> מאוסף תת-החבורות של {% equation %}G{% endequation %} אל אוסף שדות הביניים של {% equation %}E/F{% endequation %}.

בכיון השני, אנחנו לוקחים שדה ביניים {% equation %}F\subseteq K\subseteq E{% endequation %} ומתאימים לו תת-חבורה {% equation %}\Phi\left(K\right)=\left\{ \sigma\in G\ |\ \forall a\in K:\sigma a=a\right\} {% endequation %} של כל אברי {% equation %}G{% endequation %} שמשמרים את כל {% equation %}K{% endequation %}. גם פה קל להשתכנע שזו תת-חבורה: אם {% equation %}\sigma a=a{% endequation %} וגם {% equation %}\tau a=a{% endequation %} אז {% equation %}\left(\sigma\tau\right)a=\sigma\left(\tau a\right)=\sigma a=a{% endequation %} ו-{% equation %}\tau^{-1}a=\tau^{-1}\left(\tau a\right)=\left(\tau^{-1}\tau\right)a=a{% endequation %}, כך ש-{% equation %}\Phi\left(K\right){% endequation %} סגורה לכפל ולהופכי ולכן היא תת-חבורה. כעת הטענה היא ש-{% equation %}\Psi,\Phi{% endequation %} הן הופכיות זו לזו, ולכן בפרט כל אחת מהן חד-חד-ערכית ועל.

בפוסט הקודם הראיתי שאם {% equation %}E{% endequation %} הוא שדה ו-{% equation %}H{% endequation %} היא תת-חבורה כלשהי של אוטומורפיזמים של {% equation %}E{% endequation %} ואנחנו מסמנים {% equation %}K=\Psi\left(H\right){% endequation %}, אז {% equation %}E/K{% endequation %} היא הרחבת גלואה ו-{% equation %}\text{Gal}\left(E/K\right)=H{% endequation %}. הטענה הזו הגיעה בדם יזע דמעות ואלגברה לינארית. כעת, הייתי שמח לומר ש-{% equation %}\Phi\left(K\right)=\text{Gal}\left(E/K\right){% endequation %}, מה שהיה מסיים את מה שאני צריך להראות פה, אבל צריך להיזהר: ההגדרה היבשה של {% equation %}\text{Gal}\left(E/K\right){% endequation %} היא בתור {% equation %}\left\{ \sigma\in\text{Aut}\left(E\right)\ |\ \forall a\in K:\sigma a=a\right\} {% endequation %}, ואילו {% equation %}\Phi\left(K\right)=\left\{ \sigma\in G\ |\ \forall a\in K:\sigma a=a\right\} {% endequation %}. רואים את ההבדל? ב-{% equation %}\text{Gal}\left(E/K\right){% endequation %} עלולים להיות עוד איברים שלא ב-{% equation %}\Phi\left(K\right){% endequation %} כי אולי יש לנו אוטומורפיזמים של {% equation %}E{% endequation %} שמשמרים את {% equation %}K{% endequation %} אבל לא שייכים ל-{% equation %}G{% endequation %}, כלומר לא משמרים את {% equation %}F{% endequation %}. אבל הרי {% equation %}F\subseteq K{% endequation %} ולכן כל מי שמשמר את {% equation %}K{% endequation %} משמר גם את {% equation %}F{% endequation %} ושייך ל-{% equation %}G{% endequation %}, כך שכל הדקדוק הזה בקטנות נראה מיותר לגמרי - אבל אני רוצה שיהיה ברור שיש כאן משהו שצריך לומר.

יפה, אז ראינו ש-{% equation %}\Phi\left(\Psi\left(H\right)\right)=H{% endequation %}. צריך גם להראות את ההפך: {% equation %}\Psi\left(\Phi\left(K\right)\right)=K{% endequation %} לכל {% equation %}F\subseteq K\subseteq E{% endequation %}. כאן קיימת סכנה: באופן כללי, אם {% equation %}G/F{% endequation %} איננה הרחבת גלואה, בהחלט ייתכן ש-{% equation %}\Psi\left(\Phi\left(K\right)\right){% endequation %} יהיה שדה גדול יותר מ-{% equation %}K{% endequation %} - זו הסיטואציה של "אין מספיק אוטומורפיזמים של {% equation %}E{% endequation %} כדי לתפוס בדיוק את {% equation %}K{% endequation %}". בפוסט הקודם ראינו שלהגיד "הסיטואציה העגומה הזו לא מתרחשת" זה בדיוק להגיד שההרחבה שלנו היא הרחבת גלואה. נחדד:

{% equation %}E/K{% endequation %} היא הרחבת גלואה אם ורק אם השדה ש-{% equation %}\text{Aut}\left(E/K\right){% endequation %} משמרת הוא {% equation %}K{% endequation %}.

במקרה שלנו, {% equation %}\Phi\left(K\right)=\text{Aut}\left(E/K\right){% endequation %} על פי הגדרה (אברי {% equation %}\Phi\left(K\right){% endequation %} הם איברים של {% equation %}G{% endequation %}, כלומר צריכים גם לשמר את {% equation %}F{% endequation %}, אבל מכיוון ש-{% equation %}F\subseteq K{% endequation %} זה מתקיים תמיד). לכן לומר ש-{% equation %}E/K{% endequation %} היא גלואה זה לומר ש-{% equation %}\Psi\left(\Phi\left(K\right)\right)=K{% endequation %}. אז הכיוון הזה של ההוכחה מצטמצם לטענה הבאה: אם {% equation %}E/F{% endequation %} גלואה ו-{% equation %}F\subseteq K\subseteq E{% endequation %} אז {% equation %}E/K{% endequation %} גלואה. את זה אמרתי קודם, ב"יותר מכך" שבציטוט המשפט. אבל למה שזה יהיה נכון?

כאן נכנסת לתמונה העובדה שאנחנו מבינים הרחבות גלואה טוב מכמה כיוונים שונים. ראינו בפוסט הקודם ש-{% equation %}E/F{% endequation %} היא גלואה אם ורק אם {% equation %}E{% endequation %} היא <strong>שדה פיצול של פולינום ספרבילי</strong> {% equation %}p\left(x\right)\in F\left[x\right]{% endequation %}. העניין הוא ש<strong>אותו פולינום בדיוק</strong> קיים גם מעל {% equation %}K\left[x\right]{% endequation %}. כלומר, אני יכול לקחת את {% equation %}p\left(x\right){% endequation %} כפי שהוא, לשכוח מקיומו של {% equation %}F{% endequation %} ולומר "תכירו, הנה {% equation %}p\left(x\right)\in K\left[x\right]{% endequation %}, הוא פולינום ספרבילי מעל {% equation %}K{% endequation %} ושדה הפיצול שלו הוא {% equation %}E{% endequation %}". שדה הפיצול של פולינום לא משתנה כשמשנים את השם של השדה שמכיל את המקדמים, כל עוד אלו בדיוק אותם מקדמים; הפולינום בוודאי לא יפסיק להיות ספרבילי פתאום. הדבר היחיד שעלול להשתנות כשמשנים את השדה הוא ש-{% equation %}p\left(x\right){% endequation %} פתאום יפסיק להיות אי-פריק (כפי ש-{% equation %}x^{3}-2{% endequation %} האי-פריק מעל {% equation %}\mathbb{Q}{% endequation %} הופך פתאום מעל {% equation %}\mathbb{Q}\left(\sqrt[3]{2}\right){% endequation %} להיות פריק עם פירוק {% equation %}x^{3}-2=\left(x-\sqrt[3]{2}\right)\left(x^{2}+x\sqrt[3]{2}+\left(\sqrt[3]{2}\right)^{2}\right){% endequation %}). אלא ש"אי-פריק" זה לא משהו שנדרש מהפולינום כדי ששדה הפיצול שלו יהיה הרחבת גלואה, כך שזה לא רלוונטי. זה מסיים את השלב הזה של ההוכחה.

עד כה הראינו:
<ol>
 	<li>קיימת התאמה חח"ע ועל {% equation %}\Phi{% endequation %} בין שדות ביניים {% equation %}F\subseteq K\subseteq E{% endequation %} ותתי-חבורות של {% equation %}\text{Gal}\left(E/F\right){% endequation %}.</li>
 	<li>אם {% equation %}K{% endequation %} שדה ביניים שכזה אז {% equation %}E/K{% endequation %} גלואה ו-{% equation %}\text{Gal}\left(E/K\right)=\Phi\left(K\right){% endequation %}.</li>
</ol>
המסקנות המיידיות מ-2 הן שאם נסמן {% equation %}\Phi\left(K\right)=H{% endequation %} אז {% equation %}\left[E:K\right]=\left|H\right|{% endequation %} - זה נובע מכך שהרחבה היא גלואה אם ורק אם {% equation %}\left[E:K\right]=\left|\text{Aut}\left(E/K\right)\right|{% endequation %} (זו ההגדרה המקורית שלנו להרחבת גלואה). מכיוון שאנחנו יודעים ש-{% equation %}\left[E:F\right]=\left[E:K\right]\left[K:F\right]{% endequation %} (אחד המשפטים הבסיסיים בתורת השדות) ו-{% equation %}\left|G\right|=\left|G:H\right|\cdot\left|H\right|{% endequation %} (אחד המשפטים הבסיסיים בתורת החבורות) ומכיוון ש-{% equation %}G=\text{Gal}\left(E/F\right){% endequation %} גלואה ולכן {% equation %}\left|G\right|=\left[E:F\right]{% endequation %}, נקבל ש-

{% equation %}\left|G:H\right|\cdot\left|H\right|=\left[E:K\right]\left[K:F\right]{% endequation %}

ואחרי צמצום {% equation %}\left|H\right|{% endequation %} משני האגפים ({% equation %}\left[E:K\right]=\left|H\right|{% endequation %}) נקבל {% equation %}\left|G:H\right|=\left[K:F\right]{% endequation %}. זה היה ה"יותר מכך" הראשון בניסוח שנתתי למשפט.

חוב אחר שנשאר לי: להראות שאם {% equation %}K_{1}\subseteq K_{2}{% endequation %} אז {% equation %}H_{2}\subseteq H_{1}{% endequation %} (כאשר {% equation %}H_{i}=\Phi\left(K_{i}\right){% endequation %}). זה קל מאוד, כמובן: אם {% equation %}K_{1}\subseteq K_{2}{% endequation %} אז כל אוטומורפיזם של {% equation %}E{% endequation %} שמשמר את {% equation %}K_{2}{% endequation %} חייב לשמר גם את {% equation %}K_{1}{% endequation %}, ולכן כל איבר של {% equation %}\text{Gal}\left(E/K_{2}\right){% endequation %} חייב להיות גם איבר של {% equation %}\text{Gal}\left(E/K_{1}\right){% endequation %} ולכן {% equation %}H_{2}=\text{Gal}\left(E/K_{2}\right)\subseteq\text{Gal}\left(E/K_{1}\right)=H_{1}{% endequation %}.

נשאר רק החלק שהבטחתי שיהיה כבד - להראות ש-{% equation %}K/F{% endequation %} היא הרחבת גלואה אם ורק אם {% equation %}H{% endequation %} היא תת-חבורה <strong>נורמלית </strong>ב-{% equation %}G{% endequation %} ושבמקרה הזה, {% equation %}\text{Gal}\left(K/F\right)\cong G/H{% endequation %}.

כדי להבין אם {% equation %}K/F{% endequation %} היא גלואה או לא, נרצה להבין האם {% equation %}\left|\text{Aut}\left(K/F\right)\right|=\left[K:F\right]{% endequation %} או לא. מכיוון שכל המידע שיש לנו נוגע להרחבה "הגדולה" {% equation %}E/K{% endequation %}, אז כדי להבין את {% equation %}\text{Aut}\left(K/F\right){% endequation %} נרצה להסתכל על דרך לקשר אותה להרחבה הגדולה הזו. אז בואו ניקח {% equation %}\sigma\in\text{Gal}\left(E/F\right){% endequation %} וננסה להבין איך {% equation %}\sigma{% endequation %} מתנהגת כש<strong>מצמצמים</strong> אותה ל-{% equation %}K{% endequation %}, מה שאני מסמן בתור {% equation %}\sigma|_{K}{% endequation %}. אני עדיין מקבל פונקציה חח"ע שהיא הומומורפיזם של שדות; התחום שלה הוא {% equation %}K{% endequation %} והטווח שלה הוא {% equation %}E{% endequation %}, ואני גם יודע שעל {% equation %}F{% endequation %} היא הזהות, אבל <strong>זה כל מה שאני יודע</strong> באופן כללי. פונקציה כזו איננה בהכרח <strong>אוטומורפיזם</strong> של {% equation %}K{% endequation %}; בשביל זה התמונה שלה הייתה צריכה להיות בדיוק כל {% equation %}K{% endequation %}. היא מה שנקרא <strong>שיכון</strong> של {% equation %}K{% endequation %} בתוך {% equation %}E{% endequation %} - הומומורפיזם חח"ע שהתמונה שלו היא עותק ש"נראה כמו" {% equation %}K{% endequation %} בתוך {% equation %}E{% endequation %} גם אם הוא לא זהה לו.

רוצים דוגמא? בטח. זוכרים את {% equation %}\mathbb{Q}\left(\sqrt[3]{2}\right){% endequation %} ו-{% equation %}\mathbb{Q}\left(\omega\sqrt[3]{2}\right){% endequation %}? אלו שני שדות שורש שונים של הפולינום {% equation %}x^{3}-2{% endequation %} מעל {% equation %}\mathbb{Q}{% endequation %} והם בוודאי איזומורפיים, אבל הם שונים זה מזה. אני יכול לעבור מאחד לשני על ידי הפונקציה {% equation %}\sigma:\mathbb{Q}\left(\sqrt[3]{2}\right)\to\mathbb{Q}\left(\omega\sqrt[3]{2}\right){% endequation %} המוגדרת על ידי {% equation %}\sigma\left(\sqrt[3]{2}\right)=\omega\left(\sqrt[3]{2}\right){% endequation %} שעל כל {% equation %}\mathbb{Q}{% endequation %} היא הזהות. אז {% equation %}\sigma\in\text{Gal}\left(E/F\right){% endequation %} כאשר {% equation %}E{% endequation %} הוא שדה הפיצול של {% equation %}x^{3}-2{% endequation %}, אבל התמונה שלה על {% equation %}K=\mathbb{Q}\left(\sqrt[3]{2}\right){% endequation %} היא לא {% equation %}K{% endequation %} אלא עותק איזומורפי של {% equation %}K{% endequation %} ששונה מ-{% equation %}K{% endequation %}. זה ההבדל בין <strong>שיכון</strong> ובין <strong>אוטומורפיזם</strong>.

אם תסתכלו על הדוגמא האחרת, של {% equation %}\mathbb{Q}\left(\sqrt{2},\sqrt{3}\right){% endequation %}, תגלו ששם זה לא עובד - אין לנו שיכונים שאינם אוטומורפיזמים. {% equation %}\mathbb{Q}\left(\sqrt{2}\right){% endequation %} ו-{% equation %}\mathbb{Q}\left(\sqrt{3}\right){% endequation %} הם שדות <strong>לא איזומורפיים</strong>, כי באחד מהם יש פתרון למשוואה {% equation %}x^{2}-2{% endequation %} ובשני אין.

בשלב הזה אפשר אולי לנחש שכדי ש-{% equation %}K/F{% endequation %} יהיה הרחבת גלואה, התנאי שנדרש מאיתנו הוא ש<strong>כל שיכון יהיה אוטומורפיזם</strong>. כל שיכון של {% equation %}K{% endequation %} שמשמר את {% equation %}F{% endequation %}, כלומר. בואו ניתן סימונים: {% equation %}\text{Emb}\left(K/F\right){% endequation %} תהיה קבוצת השיכונים של {% equation %}K{% endequation %} שמשמרת את {% equation %}F{% endequation %}. אני אוכיח ש-{% equation %}\left|\text{Emb}\left(K/F\right)\right|=\left[K:F\right]{% endequation %}; מכיוון ש-{% equation %}\text{Aut}\left(K/F\right)\subseteq\text{Emb}\left(K/F\right){% endequation %} זה יגרור ש-{% equation %}K/F{% endequation %} גלואה (כלומר, {% equation %}\left|\text{Aut}\left(K/F\right)\right|=\left[K:F\right]{% endequation %}) אם ורק אם {% equation %}\text{Aut}\left(K/F\right)=\text{Emb}\left(K/F\right){% endequation %}.

נתחיל מכך ששיכונים ב-{% equation %}\text{Emb}\left(K/F\right){% endequation %} לא יכולים להתפרע יותר מדי - כולם מתקבלים על ידי כך שאני לוקח איבר {% equation %}\sigma\in\text{Gal}\left(E/F\right){% endequation %} ופשוט מצמצם אותו ל-{% equation %}K{% endequation %}, כלומר {% equation %}\text{Emb}\left(K/F\right)=\left\{ \sigma|_{K}\ |\ \sigma\in\text{Gal}\left(E/F\right)\right\} {% endequation %}. כשאני מבצע את הצמצום הזה חלק מהאוטומורפיזמים של {% equation %}\text{Gal}\left(E/F\right){% endequation %} עשויים להצטמצם לאותו איבר - אבל נדבר על זה בהמשך. כרגע השאלה היא <strong>למה</strong> בכלל כל שיכון של {% equation %}K{% endequation %} מתקבל בצורה כזו.

בשביל לענות על השאלה הזו צריך להיזכר במשפט שתיארתי בחטף <a href="http://gadial.net/2018/04/22/splitting_fields_algebraic_closures/">כשדיברנו על שדות פיצול</a> - שוב, אולי לא במפתיע, אחד מהמשפטים הטכניים ביותר בנושא. זה הלך כך: אם {% equation %}K_{1},K_{2}{% endequation %} שדות עם איזומורפיזם {% equation %}\sigma:K_{1}\to K_{2}{% endequation %}, ואם {% equation %}p\left(x\right)\in K_{1}\left[x\right]{% endequation %} הוא פולינום עם שדה פיצול {% equation %}E_{1}{% endequation %} ואם נסמן את שדה הפיצול של {% equation %}\sigma\left(p\right){% endequation %} (הפולינום שמתקבל מ-{% equation %}p\left(x\right){% endequation %} על ידי הפעלת {% equation %}\sigma{% endequation %} על המקדמים) ב-{% equation %}E_{2}{% endequation %} אז קיים איזומורפיזם {% equation %}\tau:E_{1}\to E_{2}{% endequation %} כך ש-{% equation %}\tau|_{K_{1}}=\sigma{% endequation %}. ההוכחה, כזכור, הייתה סוג של אינדוקציה שמתבססת על כך ש-{% equation %}E_{1}{% endequation %} מתקבל מ-{% equation %}K_{1}{% endequation %} על ידי שרשרת של הרחבות פשוטות.

במקרה שלנו {% equation %}K_{1}{% endequation %} הוא פשוט {% equation %}K{% endequation %} ואילו {% equation %}K_{2}=\sigma\left(K\right){% endequation %} - התמונה של {% equation %}K{% endequation %} תחת השיכון {% equation %}\sigma{% endequation %}. מן הסתם {% equation %}\sigma{% endequation %} הוא איזומורפיזם של שני השדות הללו כי ככה הוא נבחר מלכתחילה. כמו כן {% equation %}E_{1}=E{% endequation %} הוא שדה הפיצול של פולינום ספרבילי {% equation %}p\left(x\right)\in F\left[x\right]{% endequation %} - זו המשמעות של כך ש-{% equation %}E/F{% endequation %} היא הרחבת גלואה. אם כן, מהו {% equation %}E_{2}{% endequation %}? זה שדה הפיצול של הפולינום {% equation %}\sigma\left(p\right){% endequation %} מעל {% equation %}K_{2}{% endequation %}; אלא ש-{% equation %}\sigma\left(p\right)=p{% endequation %} כי {% equation %}p{% endequation %} הוא פולינום עם מקדמים ב-{% equation %}F{% endequation %} - בדיוק השדה ש-{% equation %}\sigma{% endequation %} משמר. לכן {% equation %}E_{2}=E_{1}{% endequation %} ואנחנו מקבלים שאפשר להרחיב את {% equation %}\sigma{% endequation %} לאוטומורפיזם {% equation %}\tau:E\to E{% endequation %}, כפי שרצינו.

הסברנו למה {% equation %}\text{Emb}\left(K/F\right)=\left\{ \sigma|_{K}\ |\ \sigma\in\text{Gal}\left(E/F\right)\right\} {% endequation %}; נשאיר להבין כמה איברים בדיוק יש ב-{% equation %}\text{Emb}\left(K/F\right){% endequation %}. לצורך כך, בואו ניקח {% equation %}\sigma_{1},\sigma_{2}\in\text{Gal}\left(E/F\right){% endequation %} וננסה להבין מתי {% equation %}\sigma_{1}|_{K}=\sigma_{2}|_{K}{% endequation %}. זה קורה אם ורק אם {% equation %}\sigma_{1}\sigma_{2}^{-1}|_{K}=\text{Id}_{K}{% endequation %}, כלומר אם ורק אם {% equation %}\sigma_{1}\sigma_{2}^{-1}\in\text{Gal}\left(E/K\right){% endequation %}. במילים אחרות: שני אוטומורפיזמים של {% equation %}E{% endequation %} נותנים את אותו שיכון של {% equation %}K{% endequation %} אם הם <strong>באותו קוסט</strong> של חבורת הגלואה של {% equation %}E/K{% endequation %}. אם ניקח נציג לכל קוסט, אז קבוצת הצמצומים של הנציגים הללו ל-{% equation %}K{% endequation %} תיתן לנו את כל {% equation %}\text{Emb}\left(K/F\right){% endequation %} (כל איבר אחר של {% equation %}E/K{% endequation %} נמצא באחד הקוסטים ולכן יתן את אותו שיכון כפי שהנציג שלו בקוסט נתן). כזכור, ראינו קודם ש-{% equation %}\left|G:H\right|=\left[K:F\right]{% endequation %} כאשר {% equation %}H=\text{Gal}\left(E/K\right){% endequation %}, והרי {% equation %}\left|G:H\right|{% endequation %} הוא בדיוק מספר הקוסטים של {% equation %}H{% endequation %} ב-{% equation %}G{% endequation %}. לכן קיבלנו ש-{% equation %}\left|\text{Emb}\left(K/F\right)\right|=\left[K:F\right]{% endequation %}, כמו שרצינו.

זה משאיר אותנו עם הטענה הבאה: {% equation %}K/F{% endequation %} היא גלואה אם ורק אם כל שיכון של {% equation %}K{% endequation %} ב-{% equation %}E{% endequation %} שמשמר את {% equation %}F{% endequation %} הוא למעשה אוטומורפיזם של {% equation %}K{% endequation %} שמשמר את {% equation %}F{% endequation %}. כלומר, אם {% equation %}\sigma\in\text{Gal}\left(E/F\right){% endequation %} אז {% equation %}\sigma\left(K\right)=K{% endequation %}.

בואו נסמן {% equation %}K^{\prime}=\sigma\left(K\right){% endequation %}. באילו סיטואציות {% equation %}K=K^{\prime}{% endequation %}? ראינו קודם שהפונקציה {% equation %}\Phi{% endequation %} שמקבלת תת-שדה של {% equation %}E{% endequation %} ומחזירה את תת-החבורה של {% equation %}G{% endequation %} שמשמרת אותו היא חד-חד-ערכית: כלומר, {% equation %}K=K^{\prime}{% endequation %} אם ורק אם {% equation %}\Phi\left(K\right)=\Phi\left(K^{\prime}\right){% endequation %}. בואו נסמן {% equation %}H=\Phi\left(K\right){% endequation %} ו-{% equation %}H^{\prime}=\Phi\left(K^{\prime}\right){% endequation %}. נשאלת כעת השאלה: אם {% equation %}K^{\prime}=\sigma\left(K\right){% endequation %} אז מהי {% equation %}H^{\prime}{% endequation %}? התשובה היא ש-{% equation %}H^{\prime}=\sigma H\sigma^{-1}{% endequation %} - <strong>ההצמדה</strong> של {% equation %}H{% endequation %} באמצעות {% equation %}\sigma{% endequation %}. זו דוגמא יפה מאוד לאופן שבו פעולת ה<strong>הצמדה</strong> בחבורות היא רלוונטית לחיים, ליקום ובכלל.

התוצאה הזו כנראה לא מפתיעה אתכם אם אתם זוכרים אלגברה לינארית, כי יש שם משהו מאוד דומה: שתי מטריצות מייצגות את אותה העתקה לינארית בבסיסים שונים אם אחת מתקבלת מהשניה על ידי הצמדה, ובמקרה הזה המטריצה המצמידה היא מטריצת המעבר מאחד הבסיסים לשני. באנלוגיה הזו, השדות אצלנו הם ה"בסיסים" ואוטומורפיזמי הגלואה הם ה"מטריצות" שמייצגות את ההעתקה הלינארית.

בואו נוכיח ש-{% equation %}H^{\prime}=\sigma H\sigma^{-1}{% endequation %}; מספיק להראות את "ההפך", כלומר את {% equation %}\sigma^{-1}H^{\prime}\sigma=H{% endequation %}, אז בואו נעשה את זה: {% equation %}\tau\in H^{\prime}{% endequation %} אם ורק אם {% equation %}\tau\left(b\right)=b{% endequation %} לכל {% equation %}b\in K^{\prime}{% endequation %}. מכיוון ש-{% equation %}K^{\prime}=\sigma\left(K\right){% endequation %} ו-{% equation %}\sigma{% endequation %} חח"ע ועל, זה מתקיים אם ורק אם {% equation %}\tau\left(\sigma\left(a\right)\right)=\sigma\left(a\right){% endequation %} לכל {% equation %}a\in K{% endequation %}. כלומר, אם ורק אם {% equation %}\tau\sigma=\sigma{% endequation %} מעל {% equation %}K{% endequation %}, כלומר אם ורק אם {% equation %}\sigma^{-1}\tau\sigma=\text{Id}_{K}{% endequation %}, כלומר אם ורק אם {% equation %}\sigma^{-1}\tau\sigma\in H{% endequation %}, כפי שרצינו.

לסיכום: קיבלנו ש-{% equation %}K/F{% endequation %} גלואה אם ורק אם {% equation %}\sigma H\sigma^{-1}=H{% endequation %} לכל {% equation %}\sigma\in G{% endequation %} - זו בדיוק ההגדרה של <strong>תת-חבורה נורמלית</strong> (טוב, אחת מההגדרות). נשאר רק לדבר על המבנה של {% equation %}\text{Gal}\left(K/F\right){% endequation %} במקרה הזה. ובכן, הבה ונגדיר הומומורפיזם <strong>של חבורות</strong> באופן הבא: {% equation %}\varphi:G\to\text{Gal}\left(K/F\right){% endequation %} מוגדר על ידי {% equation %}\varphi\left(\sigma\right)=\sigma|_{K}{% endequation %}. לא קשה לבדוק ישירות שזה אכן הומומורפיזם. מה הגרעין שלו? {% equation %}\sigma|_{K}=\text{Id}_{K}{% endequation %} אם ורק אם {% equation %}\sigma\in\text{Gal}\left(E/K\right)=H{% endequation %}. לכן נקבל ממשפט האיזומורפיזם הראשון של חבורות ש-{% equation %}G/H\cong\text{Gal}\left(K/F\right){% endequation %} - בדיוק מה שרצינו.

אתם יכולים לשאול את עצמכם, ובצדק, איפה בשלב האחרון של ההוכחה, עם משפט האיזומורפיזם הראשון, השתמשנו בכך ש-{% equation %}H{% endequation %} נורמלית. הרי {% equation %}H{% endequation %} <strong>יוצאת</strong> נורמלית מכך שהיא הגרעין של {% equation %}\varphi{% endequation %} שאני בונה שם. ובכן, אם {% equation %}H{% endequation %} לא הייתה נורמלית, אז {% equation %}\varphi{% endequation %} לא הייתה מוגדרת היטב; הטווח שלה הוא {% equation %}\text{Gal}\left(K/F\right){% endequation %} אבל כפי שראינו, אם {% equation %}H{% endequation %} לא נורמלית לא מובטח ש-{% equation %}\varphi\left(\sigma\right)=\sigma|_{K}{% endequation %} יהיה <strong>אוטומורפיזם</strong> של {% equation %}K{% endequation %}; הוא יכול להיות "סתם" שיכון.

זה מסיים את הוכחת המשפט היסודי של תורת גלואה, ובעצם מביא אותנו לסוג של נקודת שיא בסדרת הפוסטים שלנו על אלגברה מופשטת. אפשר לכאורה לעצור כאן, אבל זו למעשה הנקודה הכי גרוע לעצור בה. תורת גלואה לא נוצרה סתם בשביל היופי של לראות את הדמיון בין שדות וחבורות. היא נוצרה כדי להתמודד עם בעיות מתמטיות קונקרטיות, ואנחנו נראה דוגמאות לכך בפוסטים הבאים.
